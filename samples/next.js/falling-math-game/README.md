# Falling Math Game (Next.js)

## Overview

A client-side falling math arcade game built with Next.js App Router, React, and TypeScript. Expressions fall from the top, you type answers to clear them, and the container fills when you miss.

## Run It Locally

```bash
cd samples/next.js/falling-math-game
npm install
npm run dev
```

Open http://localhost:3000/game
