# Next.js Items List (JSX)

## Overview

This sample mirrors the React items-list-jsx example but uses the Next.js App Router. The page is rendered on the server and imports local JSON data.

## File Layout

- `samples/next.js/items-list-jsx/app/page.jsx`
  - Server component that renders the table.
- `samples/next.js/items-list-jsx/data/users.json`
  - Local data source.
- `samples/next.js/items-list-jsx/app/globals.css`
  - Styling.

## Run It Locally

```bash
cd samples/next.js/items-list-jsx
npm install
npm run dev
```
