# CSS Playground

A small static page to inspect modern CSS features and stacking contexts.

## Run

Serve the folder with any static server:

```bash
npx serve
```

Then open the printed URL (usually http://localhost:3000).

## What to look at

- Grid layout (responsive columns)
- Masonry-style layout (CSS columns fallback, grid masonry when supported)
- Stacking contexts (check DevTools → Layers to see isolated layers)
